#Audience interaction app

It's a web application you can use to create an event and while you give an online lectire the audience can participate by asking questions and upvote or downvote other viewer's questions;

#### Technologies used

##### For the back-end

- JavaScript;

	- ExpressJS;
	- Mongoose;
	- Jsonwebtoken;
	- Crypto;
	- Body-parser;

#####For the front-end;

- Angular 8
- JavaScript (TypeScript);
- HTML;
- CSS;

	- Axios;
	- Bootstrap;
